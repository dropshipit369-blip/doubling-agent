## The Agent Team

| Agent | Role | Think of them as... |
|-------|------|-------------------|
| **Scout** | Searches the web for 5 ways to double today's balance | The hustler |
| **Analyst** | Scores each play on probability, speed, risk, effort | The skeptic |
| **Strategist** | Picks the #1 play and writes the execution plan | The decision maker |
| **Chronicle** | Generates TikTok script + X post for today's flip | The storyteller |

---

## How to Run a Flip

```
> Start Flip #[N]. Current balance: $[amount]
```

The Strategist will:
1. Send Scout to research 5 opportunities
2. Send Analyst to score them
3. Pick the best play
4. Give you a step-by-step execution plan
5. Send Chronicle to generate today's content

You execute the play. Report the result. Move to the next flip.

## Rules
1. One flip per day. No skipping.
2. Every play must be executable within 24 hours.
3. No gambling, no illegal activity, no scams.
4. Real money, real results. Document everything.
5. If a flip fails, log it and adapt. Failure is content.
6. The AI picks the play. You execute. That's the deal.
