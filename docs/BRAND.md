# Brand + Positioning

## Name options
- Doubling Agent
- One Cent Agent
- Capital Ladder Agent
- Compounding Agent
- The Doubling Quest

## Recommended name
Doubling Agent

## One-line positioning
I gave an AI agent a small bankroll and one job: compound it through legal, low-risk internet opportunities, in public, with every move logged.

## Stronger positioning angle
This is not an investing guru brand.
This is a live proof-of-capability brand for agentic workers operating under real constraints.

## Audience promise
You will see:
- what agents can do
- what still needs human judgment
- what failed
- what compounded
- what can actually become a business

## Tone
- sharp
- experimental
- honest
- numbers-first
- no hype
- receipts over claims
