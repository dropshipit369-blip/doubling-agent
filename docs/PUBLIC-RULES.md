# Public Rules

## Core rule set
- Start with a fixed small bankroll.
- One active experiment at a time.
- No borrowing.
- No leverage.
- No gambling.
- No illegal or deceptive tactics.
- No autonomous spending without human approval.
- Every move must be logged.
- Every gain and loss must be recorded.
- Failures must be posted, not hidden.
- The agent can recommend, but high-impact actions require human sign-off.

## Safer framing rules
- Educational public experiment only.
- Not financial advice.
- No promises of returns.
- No solicitation to copy trades or actions.
- Focus on legal, low-risk, internet-native opportunities.

## Operating constraints
- Prefer reversible actions.
- Prefer digital-first opportunities.
- Cap downside per experiment.
- Stop an experiment if policy risk becomes unclear.
- Keep source evidence for every decision.
